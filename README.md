# alien_invasion

## 已完成功能

- 方向键控制飞船上下左右运动

- 飞船发射子弹
- 开始按钮
- 计分系统
- ~ 切换全屏和恢复
- 可以用子弹消灭外星人
- 外星人初始的方向随机
- 可以设置最大帧率
- 保存游戏记录
  - JSON保存
  - JSON存储过去游戏记录，最高分
- 显示游戏时间
  - 左上角显示时间
- 游戏重置时会把飞船重新放在屏幕最底下的中间
- 飞船碰到外星人会扣血
